Muthoot Finance Font & Icon Fixes

This tool helps you apply font and icon fixes to make your Muthoot Finance clone 100% match the original website's design.

How to Apply These Fixes

Choose one of the following methods to apply the fixes to your Muthoot Finance clone:

Method 1: Add to Each HTML File (Recommended)

Add the following code right before the closing </head> tag in each HTML file:

<!-- Font & Icon Fixes to match original website -->
<link rel="stylesheet" href="/css/font-fix.css">
<link rel="stylesheet" href="/css/icons-fix.css">

And add this script before the closing </body> tag:

<script src="/js/font-icon-fix.js"></script>

Method 2: Use Automatic Fixer Script

Add the following script before the closing </head> tag in each HTML file:

<script>
(function() {
  // Create and append font fix CSS
  var fontFixStyle = document.createElement('link');
  fontFixStyle.rel = 'stylesheet';
  fontFixStyle.href = '/css/font-fix.css';
  document.head.appendChild(fontFixStyle);
  
  // Create and append icon fix CSS
  var iconFixStyle = document.createElement('link');
  iconFixStyle.rel = 'stylesheet';
  iconFixStyle.href = '/css/icons-fix.css';
  document.head.appendChild(iconFixStyle);
  
  // Load the full fix script
  window.addEventListener('load', function() {
    var script = document.createElement('script');
    script.src = '/js/font-icon-fix.js';
    document.body.appendChild(script);
  });
})();
</script>

Method 3: Apply to All Files at Once with Node.js

If you have Node.js installed, you can create a script to automatically add the fixes to all HTML files:

  1. Create a file named apply-fixes.js with the following content:
const fs = require('fs');
const path = require('path');

// Get all HTML files recursively
function getAllHtmlFiles(dir, fileList = []) {
  const files = fs.readdirSync(dir);
  
  files.forEach(file => {
    const filePath = path.join(dir, file);
    const stat = fs.statSync(filePath);
    
    if (stat.isDirectory()) {
      getAllHtmlFiles(filePath, fileList);
    } else if (path.extname(file).toLowerCase() === '.html') {
      fileList.push(filePath);
    }
  });
  
  return fileList;
}

// Add the fix code to HTML files
function addFixesToHtmlFile(filePath) {
  let content = fs.readFileSync(filePath, 'utf8');
  
  // Head content to add
  const headContent = `


`;
  
  // Body content to add
  const bodyContent = `

`;
  
  // Add to head
  content = content.replace('', `${headContent}\n`);
  
  // Add to body
  content = content.replace('', `${bodyContent}\n`);
  
  // Write back
  fs.writeFileSync(filePath, content);
  console.log(`Added fixes to: ${filePath}`);
}

// Start process
const htmlFiles = getAllHtmlFiles('.');
console.log(`Found ${htmlFiles.length} HTML files to process.`);

htmlFiles.forEach(file => {
  addFixesToHtmlFile(file);
});

console.log('Completed adding fixes to all HTML files.');
  1. Run the script with Node.js:
node apply-fixes.js

Test the Fixes

After applying the fixes, you can test if they're working by checking these indicators:

  • The font should match the original Muthoot Finance website (Frutiger font)
  • Icons should display properly and match the original site
  • Font sizes and weights should be identical to the original

Note: If you're still seeing font or icon issues, try clearing your browser cache and refreshing the page.
