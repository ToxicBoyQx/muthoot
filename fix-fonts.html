Muthoot Finance Font Fix

This tool fixes font issues on the Muthoot Finance website by ensuring proper font loading and correcting the bold font problem.

How to Fix Font Issues

There are two ways to implement the fix:

Option 1: Add CSS to your HTML files

Copy and paste the following code into the <head> section of your HTML files:

<link rel="stylesheet" href="css/font-fix.css">

Option 2: Add this JavaScript to your pages

Copy and paste the following code right before the closing </body> tag:

<script>
// Font fixer script
document.addEventListener('DOMContentLoaded', function() {
  // Create link element
  var link = document.createElement('link');
  link.rel = 'stylesheet';
  link.type = 'text/css';
  link.href = '/css/font-fix.css';
  link.media = 'all';
  
  // Add it to the head with highest priority
  var head = document.getElementsByTagName('head')[0];
  head.appendChild(link);
  
  // Force normal weight on everything except what should be bold
  var allElements = document.getElementsByTagName('*');
  for (var i = 0; i < allElements.length; i++) {
    var el = allElements[i];
    var tagName = el.tagName.toLowerCase();
    
    if (tagName.match(/^(h1|h2|h3|h4|h5|h6|strong|b)$/i) || 
        el.classList.contains('bold') ||
        el.classList.contains('btn') ||
        el.classList.contains('head_menu')) {
      el.style.fontWeight = '600';
    } else {
      el.style.fontWeight = 'normal';
    }
  }
});
</script>
